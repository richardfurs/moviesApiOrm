# MoviesApiOrm

### Installation

Clone git repository:

```
$ git clone git@github.com:richardfurs/moviesApiOrm.git
$ cd moviesApiOrm/
```

Install Composer packages:

```
$ composer install
```

Run application:
```
$ php artisan serve
```
